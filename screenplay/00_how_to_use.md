# Video screenplay — how to use this

One tab per video: a **trailer** and **five chapters**. Each tab is a
compact shot list — *Scene N: what's on screen + the action*. No talk
track; you narrate each scene in your own words.

Conventions used in every tab:
- **[SCREEN]** what the viewer sees · **[DO]** the on-camera action ·
  **[CUT]** an edit instruction (hard cut / speed-ramp / hide) ·
  **[★]** the hero moment — the shot the video exists for.
- Target 3–4 min per chapter, trailer ≤ 90s.
- Open and close every chapter on the **app landing page** (home base).
- Prep off-camera: build the .xlsm, pre-warm Genie, pre-create jobs/spaces.
  Nothing should appear on screen "from nowhere".
- Record order: Ch1 → Ch3 → Ch5 → Ch2 → Ch4. Cut the trailer last.

All data synthetic; no customer names on screen.
